# Digimon Battle
This is a digimon turn-based game made using Scala, ScalaFX and SceneBuilder. 

## Prerequisite
- JDK: 1.8.0
- Scala: 2.12.17
- SBT 

## How to Start
Run the following command in the terminal to start the game
```
sbt run
```
